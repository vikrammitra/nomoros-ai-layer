### 🧠 Processing Logic (step-by-step)

Implement the endpoint with the following logic:

#### 1. Validate inputs

* Confirm `document_id` exists
* Confirm document has:

  * `text_extracted = true`
  * `document_type = "TA6"`
* If not, return HTTP 400 with clear error message

#### 2. Load OCR text

* Retrieve full extracted text from storage (DB or blob store)
* Preserve character offsets (do not re-chunk yet)

#### 3. Prepare TA6 LLM prompt

* Use a **system + user prompt** structure
* Enforce:

  * JSON-only output
  * No hallucination
  * Evidence quotes with char offsets
* The LLM prompt must:

  * Confirm the document is a TA6
  * Extract structured sections
  * Identify contradictions
  * Identify missing or referenced attachments
  * Generate follow-up questions

(Assume the TA6 prompt text is stored in a constant or template file.)

#### 4. Call Azure OpenAI

* Use existing Azure OpenAI client
* Use:

  * `temperature = 0`
  * `response_format = json`
* Pass full extracted text (truncate safely if token limits are exceeded)

#### 5. Validate LLM output

* Parse JSON
* Validate against a Pydantic model:

  * `TA6ExtractionResult`
* If validation fails:

  * Mark document status = `FAILED`
  * Persist error message

#### 6. Persist results

Store:

* `ta6_extraction_json`
* `parsed_at`
* `parser_version`
* Update document status to:

  ```text
  PARSED_TA6
  ```

#### 7. Return response

Return:

```json
{
  "document_id": "...",
  "document_type": "TA6",
  "status": "PARSED_TA6",
  "ta6_extraction": { ... }
}
```

### 🧱 Data Models to Add (Pydantic)

Create the following models:

```python
class Evidence(BaseModel):
    ref: Optional[str]
    quote: str
    char_start: int
    char_end: int

class TA6Question(BaseModel):
    question_id: Optional[str]
    question_text: str
    answer_normalised: Literal[
        "yes", "no", "unknown", "not_applicable", "details_provided"
    ]
    answer_raw: Optional[str]
    details: Optional[str]
    answer_status: Literal["clear", "unclear", "missing", "contradictory"]
    clarification_question: Optional[str]
    evidence: List[Evidence]

class TA6Section(BaseModel):
    section_name: str
    questions: List[TA6Question]

class TA6Contradiction(BaseModel):
    type: str
    description: str
    items: List[dict]

class TA6FollowUp(BaseModel):
    priority: Literal["high", "medium", "low"]
    question: str
    reason: Literal[
        "missing", "unclear", "contradiction", "attachment_referenced"
    ]

class TA6ExtractionResult(BaseModel):
    is_ta6: bool
    document_meta: dict
    sections: List[TA6Section]
    contradictions: List[TA6Contradiction]
    missing_or_referenced_attachments: List[dict]
    follow_up_questions: List[TA6FollowUp]
```
