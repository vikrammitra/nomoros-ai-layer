🧠 CHUNKING STRATEGY (MANDATORY)
Why

TA6 forms:

Are long (30+ pages)

Have repeated headers/footers

Contain checkbox noise

Will exceed token limits if processed in one pass

Strategy to implement

Step 1 – Normalise text

Remove repeated page headers/footers (e.g. “Law Society TA6 (5th edition)”)

Preserve character offsets after cleaning

Step 2 – Chunking
Chunk by:

~3,000–3,500 characters

With 300 char overlap

Do NOT split mid-question if possible (best-effort)

Each chunk should include:

{
  "chunk_id": int,
  "text": str,
  "char_start": int,
  "char_end": int
}


Step 3 – Map phase

Run TA6 extraction prompt on each chunk

Output: partial TA6 JSON (same schema, but sparse)

Step 4 – Reduce phase

Merge sections/questions by:

section_name

question_id OR question_text

De-duplicate evidence

Merge contradictions

Re-rank follow-up questions by priority

🧱 PYDANTIC MODELS (ADD EXACTLY)
from typing import List, Optional, Literal
from pydantic import BaseModel

class Evidence(BaseModel):
    ref: Optional[str]
    quote: str
    char_start: int
    char_end: int

class TA6Question(BaseModel):
    question_id: Optional[str]
    question_text: str
    answer_normalised: Literal[
        "yes", "no", "unknown", "not_applicable", "details_provided"
    ]
    answer_raw: Optional[str]
    details: Optional[str]
    answer_status: Literal[
        "clear", "unclear", "missing", "contradictory"
    ]
    clarification_question: Optional[str]
    evidence: List[Evidence]

class TA6Section(BaseModel):
    section_name: str
    questions: List[TA6Question]

class TA6Contradiction(BaseModel):
    type: str
    description: str
    items: List[dict]

class TA6FollowUp(BaseModel):
    priority: Literal["high", "medium", "low"]
    question: str
    reason: Literal[
        "missing", "unclear", "contradiction", "attachment_referenced"
    ]

class TA6ExtractionResult(BaseModel):
    is_ta6: bool
    document_meta: dict
    sections: List[TA6Section]
    contradictions: List[TA6Contradiction]
    missing_or_referenced_attachments: List[dict]
    follow_up_questions: List[TA6FollowUp]
